Midterm project Readme file

This project leverages the following open APIs and packages
    Flask Web App
    NLTK python package for natural language interpretaion
    Vader Sentiment Analysis
    Spacy API to find Nouns in the given sentence

This project consists of the following code components
    Python program driving the flask api
    HTML files and templates to support different decorator outputs
    Hosted on Amazon EC2 for running this as an open service for both the functions - Sentence sentiment analysis and Finding nouns in a given sentence
